## How to Use

1. Open https://freetexttospeech.pages.dev in a modern web browser (Chrome, Firefox, Safari, Edge)
2. Enter or paste your text in the text area
3. Select a voice from the dropdown
4. Adjust speed, pitch, and volume using the sliders
5. Click "Speak" to start
6. Use pause/resume/stop buttons to control playback

## Keyboard Shortcuts

- `Ctrl/Cmd + Enter` - Start speaking
- `Escape` - Stop speaking

## Browser Compatibility

The Web Speech API is supported in:
- Chrome/Edge (Chromium) - Full support
- Safari - Full support
- Firefox - Full support

## Technical Details

This website uses the [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API), specifically the `SpeechSynthesis` interface. All text-to-speech processing happens locally in your browser - no data is sent to any servers.
